# Week 1 Submissions — Setup Challenge

**Challenge:** Generate a colorful, one-file HTML Tetris game using a local AI inference engine.  
Full details → [CHALLENGES.md](../../CHALLENGES.md#-week-1--setup-challenge)

---

## What to Submit

Create a folder: `submissions/week-1/your-github-username/`

Include:
- `tetris.html` — your generated Tetris game (single file, must run in a browser)
- `NOTES.md` — completed from the template below

---

---

## Validation

Your submission will be marked:
- 👍 Validated — Tetris runs in a browser, generated by a local model
- 👎 Not validated — with a note on what's missing

> If your hardware couldn't complete the challenge, submit anyway with a completed NOTES.md explaining what you tried and where it stopped. That outcome is equally valuable data.
