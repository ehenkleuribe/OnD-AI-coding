# Week 3 Submissions — Automation & Scripting Challenge

**Challenge:** Build a runnable CLI script or automation tool that solves a real task in your daily work.  
Full details → [CHALLENGES.md](../../CHALLENGES.md#-week-3--automation--scripting-challenge)

## What to Submit

Create a folder: `submissions/week-3/your-github-username/`

Include:
- Your script / tool files
- A `README.md` *inside your folder* explaining what it does and how to run it (this is part of the deliverable)
- `NOTES.md` — completed from the template below
